Discord

Discord commands

Important

All servers has default Prefix <!> , In case that ou changed your prefix,use your new prefix to commands

Moderation
# Usage description Permission
1 !filter banphrase add "Word" Add word to the blacklist Administrator
2 !dj role set "ROLE Name" set a custom role to have dj's permissions Administrator
3 !dm set "Message" set dm for your server. Can use Custom Variables like: <{UserDM}> and <{GuildDM} > Owner
4 !twitch set channel "CHANNEL NAME" link your twitch channel to the bot Owner
5 !twitch set dm "MESSAGE" Message that will be send when user starts streaming Owner
6 !cmd create "Name" "Content" Create custom cmd Administrator
7 !cmd remove "Name" remove cmd Administrator

Important

If you have run <!Dj set "ROLE NAME">,music system will only work for people that has this Role!

Fun and others
# Usage description Permission

Twitch

Twitch commands

Important

To use<!Title set "Message"> , you need to add permission channel editor to botbravery

Moderation
# Usage description Permission
1 !filter banphrase add "Word" Add word to the blacklist Moderator
2 !Title set "MESSAGE" Change title stats Moderator
3 !subs dm set "Message" set dm when user subscribed . Can use Custom Variables like: <{UserName}> and <{ChannelName} > Broadcaster
4 !cmd create "Name" "Content" Create custom cmd Moderator
5 !cmd time set "Name" "Minutes" Send command every X Minutes Moderator
6 !cmd remove "Name" remove cmd Moderator
7 !filter banphrase remove "Word" Aremove word from the blacklist Moderator
8 !domain permit add "domain or link" permit user to use domain or link Moderator
9 !domain permit remove "domain or link" remove domain or link Moderator
10 !discord sync "Discord User ID" Link your twitch channel with discord user Broadcaster
